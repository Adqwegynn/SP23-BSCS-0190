HR Virtual Assistant - CV Matching System

//What Does It Do?
Analyzes how well a candidate's CV matches a job description and provides
(1) Match percentage (0-100%)
(2) Matched and missing skills
(3) Experience comparison
(4) Hiring recommendations
(5) Downloadable report
====================================================================
Key Features:
It supports all PDF, DOC and DOCX Files.
It ensures that data is secure and not sent to external servers
No API required - It is cost Free 
It has the capability of auto clean up 
And it is secure because all processing will happen in your browser
====================================================================
SETUP:
To setup I have used index.html and used HTML for Main Interface 
Then I have used style.css and used CSS for styling the website
Then I have used app.js for logic and security 
=================================================================
Privacy & Security:
Your data is 100% safe:

All processing happens locally in your browser
No data sent to external servers
CV content never stored or saved
Automatic cleanup after 5 minutes
No tracking, cookies, or analytics
GDPR compliant
